    Java List Interview Questions

    1. What is the difference between List and Set in Java?

    Answer: A List allows duplicate elements and maintains the order of insertion, while a Set does not allow duplicate elements and does not guarantee any specific order.


    2. What are the main implementations of the List interface in Java?

    Answer: The main implementations of the List interface are ArrayList, LinkedList, and Vector.


    3. What is the difference between ArrayList and LinkedList?

    Answer:

    • ArrayList: Uses a dynamic array to store elements. Provides fast random access and is better for storing and accessing data. However, adding and removing elements can be slow as it requires shifting elements.
    • LinkedList: Uses a doubly linked list to store elements. Provides faster insertion and deletion but slower access time compared to ArrayList because it requires traversing the list.

    4. What is the difference between Vector and ArrayList?

    Answer:

    • Vector: Is synchronized, meaning it is thread-safe. It is generally slower than ArrayList due to synchronization overhead. It has been largely replaced by ArrayList in new code.
    • ArrayList: Is not synchronized, making it faster than Vector for non-thread-safe scenarios.

    5. How do you create an ArrayList in Java?

    Answer: You can create an ArrayList by using the following code:

    import java.util.ArrayList; public class ArrayListExample { public static void main(String[] args) { ArrayList<String> list = new ArrayList<>(); list.add("Apple"); list.add("Banana"); System.out.println("List: " + list); } }

    6. How do you create a LinkedList in Java?

    Answer: You can create a LinkedList by using the following code:

    import java.util.LinkedList; public class LinkedListExample { public static void main(String[] args) { LinkedList<String> list = new LinkedList<>(); list.add("Apple"); list.add("Banana"); System.out.println("List: " + list); } }

    7. How do you create a Vector in Java?

    Answer: You can create a Vector by using the following code:

    import java.util.Vector; public class VectorExample { public static void main(String[] args) { Vector<String> vector = new Vector<>(); vector.add("Apple"); vector.add("Banana"); System.out.println("Vector: " + vector); } }

    8. How does ArrayList handle resizing?

    Answer: ArrayList automatically resizes itself when elements are added beyond its initial capacity. It increases the size of the underlying array by 50% each time it needs more space.


    9. How does LinkedList handle insertion and deletion?

    Answer: LinkedList handles insertion and deletion efficiently by adjusting the links between nodes. This allows it to perform these operations in constant time, given a reference to the node.


    10. What is the default initial capacity of a Vector?

    Answer: The default initial capacity of a Vector is 10. If more elements are added, the Vector will grow as needed, typically doubling its size.


    11. How can you synchronize an ArrayList?

    Answer: You can synchronize an ArrayList by using Collections.synchronizedList:

    import java.util.ArrayList; import java.util.Collections; public class SynchronizedArrayListExample { public static void main(String[] args) { ArrayList<String> list = new ArrayList<>(); List synchronizedList = Collections.synchronizedList(list); } }

    12. How does LinkedList implement the Queue interface?

    Answer: LinkedList implements the Queue interface by providing methods such as offer, poll, and peek, which are used for queue operations.


    13. Can Vector be used in a multi-threaded environment?

    Answer: Yes, Vector is synchronized and can be safely used in a multi-threaded environment. However, for better performance, consider using Collections.synchronizedList with ArrayList or other concurrent collections.


    14. How do you convert a LinkedList to an Array?

    Answer: You can convert a LinkedList to an array using the toArray method:

    import java.util.LinkedList; public class LinkedListToArrayExample { public static void main(String[] args) { LinkedList<String> list = new LinkedList<>(); list.add("Apple"); list.add("Banana"); String[] array = list.toArray(new String[list.size()]); System.out.println("Array: " + array[0] + ", " + array[1]); } }

    16. How do you remove duplicates from an ArrayList?

    Answer: You can remove duplicates from an ArrayList by using a Set:

    import java.util.ArrayList; import java.util.HashSet; import java.util.Set; public class RemoveDuplicatesExample { public static void main(String[] args) { ArrayList<String> list = new ArrayList<>(); list.add("Apple"); list.add("Banana"); list.add("Apple"); Set<String> set = new HashSet<>(list); ArrayList<String> uniqueList = new ArrayList<>(set); System.out.println("List without duplicates: " + uniqueList); } }

    17. What are the advantages of using LinkedList over ArrayList?

    Answer:

    • Efficient Insertions/Deletions: LinkedList provides constant time complexity for insertion and deletion at both ends of the list.
    • No Resizing Overhead: LinkedList does not require resizing like ArrayList does.

    18. What are the differences between the add() and addAll() methods in ArrayList?

    Answer:

    • add(E e): Adds a single element to the end of the list.
    • addAll(Collection c): Adds all elements from the specified collection to the end of the list.
